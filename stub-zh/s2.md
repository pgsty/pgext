`s2` `0.0.2` 版是 Python `s2sphere` 软件包的 PL/Python 封装。它把经纬度转换为有符号 `bigint` S2 单元标识或文本令牌，也能反向转换，并提供有效性、层级、父子、相邻、包含及相交辅助函数。

### 示例

先安装与 PostgreSQL 匹配的 `plpython3u` 软件包，并让服务器 Python 运行时可以导入 `s2sphere`，然后执行：

```sql
CREATE LANGUAGE plpython3u;
CREATE EXTENSION s2;
SELECT s2_cellid_from_latlng(10.1234, -72.1234);
SELECT s2_token_from_latlng(10.1234, -72.1234);
```

尽管目录把它标为纯 SQL，每个功能入口实际都会执行不可信 PL/Python 并导入宿主机 Python 依赖。只有超级用户才应安装或控制 `plpython3u`；必须固定并审计具体 Python 环境。2018 年的仓库没有当前 PostgreSQL、Python 或 `s2sphere` 兼容矩阵。S2 单元是分层近似；若正确性依赖真实空间边界，应再使用精确几何谓词。
