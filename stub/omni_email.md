


## Usage

> [omni_email: E-mail framework](https://docs.omnigres.org/omni_email/reference/)

The `omni_email` extension provides an e-mail framework for PostgreSQL. It depends on `omni_id`, `omni_cloudevents`, and `omni_polyfill`.

The extension provides types and functions for composing, storing, and managing email messages within the database, integrating with the Omnigres CloudEvents system for event-driven email workflows.
