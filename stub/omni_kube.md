


## Usage

> [omni_kube: Kubernetes (k8s) integration](https://docs.omnigres.org/omni_kube/api/)

The `omni_kube` extension provides Kubernetes API integration from PostgreSQL. It depends on `omni_httpc`, `omni_web`, `omni_var`, and `omni_yaml`.

The extension enables interaction with Kubernetes clusters directly from SQL, allowing you to query and manage Kubernetes resources using the K8s API through the HTTP client provided by `omni_httpc`.
