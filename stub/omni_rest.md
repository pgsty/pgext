


## Usage

> [omni_rest: REST API toolkit (with PostgREST support)](https://docs.omnigres.org/omni_rest/protocols/)

The `omni_rest` extension provides REST API functionality with basic PostgREST protocol compatibility.

The extension implements foundational support for the PostgREST protocol. Refer to the [PostgREST documentation](https://postgrest.org/) for protocol details.

This is an early release; not all PostgREST functionality is implemented yet, with essential use cases prioritized. Depends on `omni_httpd`.
