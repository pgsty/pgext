## Install

Make sure [**PGDG**](/repo/pgdg) and [**PIGSTY**](/repo/pgsql) repo available:

```bash
pig repo add pgdg pigsty -u   # add both repo and update cache
```

[**Install**](https://ext.pgsty.com/usage/install) this extension with [**pig**](/pig):

```bash
pig install hunspell_ru_ru_aot;		# install via package name, for the active PG version

pig install hunspell_ru_ru_aot -v 18;   # install for PG 18
pig install hunspell_ru_ru_aot -v 17;   # install for PG 17
pig install hunspell_ru_ru_aot -v 16;   # install for PG 16
pig install hunspell_ru_ru_aot -v 15;   # install for PG 15
pig install hunspell_ru_ru_aot -v 14;   # install for PG 14
pig install hunspell_ru_ru_aot -v 13;   # install for PG 13

```


[**Create**](https://ext.pgsty.com/usage/create) this extension with:

```sql
CREATE EXTENSION hunspell_ru_ru_aot;
```
